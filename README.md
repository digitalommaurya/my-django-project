# 📚 Django Study Notes & PDF Resources

A curated collection of study materials, hand-written notes, and PDF guides for mastering **Django** and **Python Web Development**.

This repository contains comprehensive notes covering the following Django topics:

- **MVC/MVT Architecture**: Understanding Models, Views, and Templates.
- **ORM & Databases**: Querying, Migrations, and Database relationships.
- **Django Rest Framework (DRF)**: Building APIs.
- **Deployment**: Notes on render.
